# 新的 Apt 和 Yum 源

1. 如果您想获得最新的 Docker，只需更新您的 Docker 仓库源列表就可以了。
2. 我们还为 rpm 的 yum 仓库源。
3. 每个人都可以获得动态的二进制

现在，我们可以非常欣喜的告诉大家我们已经为 Docker 的安装创建并部署了新的 apt 和 yum 源。考虑到不再使用 lxc 作为默认的 Docker 后端程序，您无需再去执行 `apt get install lxc docker`，让我们进一步了解一下新的仓库吧。
